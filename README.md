## C# Tutorials

Bu repo, C# konularını örnek kodlarla pekiştirmek için oluşturulmuş konsol uygulaması projelerinden oluşur.

## Klasörler

Repo içeriği; her konu için ayrı bir proje klasörü ve ona bağlı tutorial dokümanı şeklindedir.

- `1_BasicsOfCsharp/` : Temel C# yapı taşları (türler, değişkenler, sabitler/enums, operatörler, koşullar, döngüler, basit IO).
  - Doküman: `1_BasicsOfCsharp/BasicsOfCsharp.md`
- `2_MethodsAndFunctions/` : Metotlar (functions); çağırma, parametre/return, overload, `ref/out`, optional parametre, local function ve expression-bodied.
  - Doküman: `2_MethodsAndFunctions/MethodsAndFunctions.md`

## Çalıştırma

Her örneği çalıştırmak için ilgili proje klasörüne girip:

```bash
dotnet run
```

Not: Bazı projelerde `Program.cs` içinde ilgili `Run()` çağrıları yorum satırı olarak bırakılmış olabilir. Çalıştırmak istediğiniz örneğin çağrısını ilgili dosyada aktif hale getirin.
