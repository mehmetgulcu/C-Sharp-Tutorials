## C# Tutorials

Bu repo, C# konularını örnek kodlarla pekiştirmek için oluşturulmuş konsol uygulaması projelerinden oluşur.

### Yapı

- `1_BasicsOfCsharp/` : Temel C# yapı taşları (türler, değişkenler, sabitler/enums, operatörler, koşullar, döngüler, basit IO).

### Çalıştırma

Örnekleri çalıştırmak için ilgili projenin klasörüne girip:

```bash
dotnet run
```

> Not: Bazı örneklerde `Program.cs` içinde ilgili `Run()` çağrıları yorum satırı olarak bırakılmış olabilir. Çalıştırmak istediğiniz örneğin çağrısını ilgili dosyada aktif hale getirin.
