## Çalıştırma

Her örneği çalıştırmak için ilgili proje klasörüne girip:

```bash
dotnet run
```

Not: Bazı projelerde `Program.cs` içinde ilgili `Run()` çağrıları yorum satırı olarak bırakılmış olabilir. Çalıştırmak istediğiniz örneğin çağrısını ilgili dosyada aktif hale getirin.
