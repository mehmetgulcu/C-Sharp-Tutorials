## BasicsOfCsharp

`1_BasicsOfCsharp` projesi, C# dilinin temel yapı taşlarını konsol uygulaması üzerinden örnekler:

- `1_Basics.cs` : Temel veri türleri (int, string, bool, decimal, double, char)
- `2_ConstsAndEnums.cs` : `const` ile sabitler ve enum kullanımı
- `3_Operators.cs` : Aritmetik, karşılaştırma ve mantıksal operatörler
- `4_Conditional_Statements.cs` : `if/else` ve `switch` akışları
- `5_Loops.cs` : `for`, ters `for`, `do-while` ve `while` döngüleri
- `6_Input_Output.cs` : Kullanıcıdan `Console.ReadLine()` ile giriş alma ve temel IO

### Önemli Dosyalar

- `Program.cs` : Çalıştırılacak örneği seçmek için ilgili `Run()` metot çağrılarını içerir (bazıları yorum satırı olabilir).
- `1_BasicsOfCsharp.csproj` : `net9.0` hedefine sahip konsol projesi.

### Nasıl Çalıştırılır

1. `1_BasicsOfCsharp/` klasörüne girin.
2. `Program.cs` içinden istediğiniz örneğin çağrısını yorumdan çıkarın.
3. Projeyi çalıştırın:

```bash
dotnet run
```

### Hızlı Kontrol Listesi

Çıktıyı görmeniz için ilgili `Run()` çağrısının aktif olması gerekir:

- `Basics.Run()` -> `1_Basics.cs`
- `ConstsAndEnums.Run()` -> `2_ConstsAndEnums.cs`
- `Operators.Run()` -> `3_Operators.cs`
- `ConditionalStatements.Run()` -> `4_Conditional_Statements.cs`
- `Loops.Run()` -> `5_Loops.cs`
- `InputOutput.Run()` -> `6_Input_Output.cs`
